# 3. CVE Agent (agents/cve_agent.py)

### Model  
`gemini-2.0-flash` (Google Gemini)

### Steps  
1. LLM extracts product keywords  
2. Calls NVD client:  
   - `search_cves(keyword)`  
3. Returns multiple CVEs per keyword:

```
id, cvss, description, source_keyword, confidence
```

100% real data.
